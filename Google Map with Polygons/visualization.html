Google Map Visualization

There are several examples that can demonstrate possibilities of map visualization. The first example is the map of US with rendering bounds of states and filling states by different colors.

If you double click on the following states: Alabama, Arkansas, Arizona, California you can see details of state. To return previous view refresh page.

The Google map provides possibilities to add on the map different objects. The next example is adding the pie chart to map.

You can mouse over the parts of pie and get additional information.

So we can add any google chart to map. For example donuts charts for US.
